## Usage
To use this tool, simply run it and provide the path to a text file containing a list of URLs as input. The tool will then process the URLs and perform its intended function.

## Example

### Windows

```powershell
./DomainsGrabber.exe
```

### Linux

```bash
./DomainsGrabber
```
